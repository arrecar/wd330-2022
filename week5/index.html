  2. The Locked Box

    3. Consider the following (rather contrived) object:

    const box = {
            locked: true,
            unlock() { this.locked = false; },
            lock() { this.locked = true;  },
            _content: [],
            get content() {
              if (this.locked) throw new Error("Locked!");
              return this._content;
            }
          };

    It is a box with a lock. There is an array in the box, but you can get at it only when the box is unlocked. Directly 4 accessing the private _content property is forbidden.
